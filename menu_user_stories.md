# User Stories for Menu Navigation

---

## User Story 1: Access Menu Options

**Title:** Access Menu Options  
_As a user, I want to access a menu with options for configuring my habits, viewing reports, editing my profile, and signing out, so that I can easily navigate to different parts of the app._

**Acceptance Criteria:**
1. Menu is visible on all main pages
2. Menu contains options: Habits, Reports, Profile, Sign Out
3. Menu is easy to access and use

**Priority:** High  
**Story Points:** 5  

**Notes:**
- Menu should be responsive (mobile + desktop)

---

## User Story 2: Navigate to Profile Page

**Title:** Navigate to Profile  
_As a user, I want to navigate to my profile page from the menu, so that I can view and edit my personal information._

**Acceptance Criteria:**
1. Clicking "Profile" opens profile page
2. User details are displayed correctly
3. User can edit and save changes

**Priority:** Medium  
**Story Points:** 3  

**Notes:**
- Ensure user data loads correctly

---

## User Story 3: Navigate to Habits Page

**Title:** Navigate to Habits Page  
_As a user, I want to access the habits page from the menu, so that I can configure and manage my habits._

**Acceptance Criteria:**
1. Clicking "Habits" opens habits page
2. User can add/edit/delete habits
3. Changes are reflected immediately

**Priority:** High  
**Story Points:** 5  

**Notes:**
- Core feature of the application

---

## User Story 4: Sign Out from Menu

**Title:** Sign Out  
_As a user, I want to sign out of my account using an option in the menu, so that I can securely log out when I'm finished using the app._

**Acceptance Criteria:**
1. Clicking "Sign Out" logs the user out
2. User is redirected to login page
3. Session is cleared after logout

**Priority:** High  
**Story Points:** 3  

**Notes:**
- Important for security
